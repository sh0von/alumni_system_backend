API Documentation

1. Signup

Registers a new alumni user.

Method: POST

URL: /signup

Request Body:

  {
    "first_name": "John",
    "last_name": "Doe",
    "username": "johndoe",
    "email": "johndoe@example.com",
    "password": "password123"
  }

Response:

  {
    "message": "Alumni registered successfully"
  }

2. Login

Logs in an existing alumni user.

Method: POST

URL: /login

Request Body:

  {
    "username": "johndoe",
    "password": "password123"
  }

Response:

  {
    "message": "Login successful",
    "token": ""
  }
